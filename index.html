My work

Hotel Laravel

Web application using Laravel with the theme being a hotel. User types: User, admin and manager.
- Users have the option to access the main page (login needed). From there they have access to the following options: renting a room, adding items to their minifridge and adding additional services. They can pay the bill directly from the website.
- Admins can deal with CRUD operations on the "User" role (they are able to change the user type). This page is protected which means that anyone who tries to get on to the page that is not an Admin will be redirected to an another page.
- Managers have the ability to overview and do CRUD operations on hotel reservations, services, rooms, minibar and hotel employees.

Posty

Posty is my social media test project done in PHP Laravel. It has these features: posting, liking, un-liking, commenting. It has an implemented like counter and set of rules which are making it work well. In order to do any of the features, users have to login. Some of the defensive mechanisms are: Users cannot like or dislike a post multiple times, Users can delete their own posts but cannot delete the posts of others. Every post has a like and dislike counter. By clicking on the users name you can view their profile which contains information such as: All of their posts, number of posts, number of likes, their email and username.


Elearning

College group project using ASP.NET Core. This project has been done with 3 of my colleagues. The theme of the project is an IT course website using a NOSql database. The website has user roles (normal user and admin user). The user has an option to view the list of courses that they can further check out if the details button has been clicked (Each course also has a category it belongs to). Each course has a school it belongs to, because of this the user has an option to search schools and then look at their individual courses. The project also has specialized program which is a feature that lists IT courses from beginner to expert in order of most efficient learning (This serves as a guide to new students in mastering certain IT fields). The admin user has its own specialized admin panel where they can do CRUD operations on the normal users, schools, courses and categories.
